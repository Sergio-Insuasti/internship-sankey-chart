# internship-sankey-chart
## Developed by Sergio Insuasti

A Data Analytics Project which shows my Internship/Grad application cycle for 2026. 

Welcome to an analysis of my Internship/Grad application season for 2026!

### About
While my journey so far has come to a positive conclusion, I am sure other
developers would want to figure out how each of their applications have
"travelled" down their respective application pipelines.

### The Program
In doing such, I developed a quick program which (given a csv file I've templated)
can analyse and provide Data Analytics and insights to the following:
- Number of Applications Submitted
- Number of Applications progressed to:
  - Online Assessments
  - Interviews
  - Final Rounds
  - Offers Sent
  - Offers Accepted
- Industries of Companies (fields colloquialised)
- Average Salary
